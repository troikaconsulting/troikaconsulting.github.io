Past Projects

NGB Markets, Inc.

NGB Markets brought us a data extraction problem that would a learning algorithm to get the job done efficiently. Receiving hundreds of spreadsheets a day, they previously had to manually hardcode templates for extracting data from spreadsheets that all contained similar information in strikingly different formats . We created a system that would automatically pull in emailed spreadsheets, parse them, extract line-items using our inferencing module and deduce what USDA labeled food item it corresponded to. This data exraction and classification problem required a full stack system starting from the second a spreadsheet is emailed to NGB, to the point in which they update their database with the relevant data -- all without the constraints of fine-tuning for a particular set of spreadsheet formats.
